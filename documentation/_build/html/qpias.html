class qpias.game.Game(dpi=96, width=800, height=600, dt=0.0001, fps=60)¶

Bases: object

Creates the game window and stores all the information about the game state.

Parameters:

  • dpi (int, optional) – Dots-per-inch, default 96

  • width (int, optional) – Screen width in pixels, default 800

  • height (int, optional) – Screen height

  • dt (float, optional) – Initial time step for animation, default 1e-4

  • fps (int, optional) – Frames-per-second, default 60

  • superposition_mode (bool, optional) – Whether to display a single wave function or a superposition of wave functions, default False (single wave function)

  • eigenvectors_mode (bool, optional) – Whether to display all the energy eigenvectors or the current wave function, default False (current wave function)

Example:

>>> import qpias
>>> game = qpias.game.Game(width=1920, height=1080)
>>> game.quit()

generate_basis_functions()¶

Creates a basis set based on the solutions to the one-dimensional particle-in-a-box without a potential.

Stores results in Particle.basis and Particle.momenta.

The basis set in position space is defined as

\[\chi_n(x) = \sqrt{\frac{2}{L}} \sin\left( \frac{n\pi x}{L} \right) \:\:\: n=1,2,\cdots,n_\text{max}\]

The basis set in momentum space is defined as

\[\chi_n(k) = \frac{k\pi}{L} \:\:\: k=\pm 1,\pm 2,\cdots,\pm n_\text{max}\]
generate_hamiltonian(potential=None)¶

Creates the hamiltonian matrix elements. Stores the information in Particle.H.

Parameters:

potential (numpy.ndarray, optional) – Potential to use, default None

See also, generate_potential_matrix().

\[\mathbf{H}_{mn} = \mathbf{T}_{mn} + \mathbf{V}_{mn}\]

The kinetic energy matrix is calculated as:

\[\begin{split}\mathbf{T}_{mn} = \begin{cases}\frac{n^2 \times \pi^2} {2 \times m \times L^2} & \text{if } n = m \\ 0 & \text{if } n \neq m\end{cases}\end{split}\]

position_momentum_collapse(psi, x0=None, k0=None, momentum=False)¶

Collapse the wave function to a particle with a position/momentum with minimum uncertainty: \(\Delta_x \times \Delta_p = 1/2\) For a position collapse: \(\Delta_x = L / 200\) For a momentum collapse: \(\Delta_p = 5 / L\)

Returns:

Tuple of the new wave function coefficients, the collapsed position \(x_0\) and the collaposed momentum \(p_0\).
